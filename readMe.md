## Oversikt

Dette prosjektet er en skoleoppgave.  
Målet er å øve på:

- JavaScript (For loops, if, array)

## Oppgavebeskrivelse
Lag en nettside med et enkelt søkefelt hvor man kan skrive inn et navn (fornavn, etternavn eller begge deler). Når man trykker på en knapp og «Enter», skal informasjon om personen vises på skjermen.

Krav til siden:
- Input felt man kan søke opp med
- Søke med både Button og Enter
- Minst 5 karakterer/personer i Array
- Dekorere med CSS
